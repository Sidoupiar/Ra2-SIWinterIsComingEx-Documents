# 属性-国家

## 出厂经验值

类似于原版的国家能力，可以指定一些单位出厂即可获得经验值和升级。

位于 `rulesmd.ini`：

```ini
[SomeCountryType]
; 建筑的出厂经验值
; 以下这三个列表的元素数量需要相匹配
EXPInit.Types.Building=                         ; 建筑列表 , 这些单位获得经验值 ， 默认值是 空
EXPInit.Amounts.Building=0                      ; 浮点数列表 , 获得这个数量的经验值 ， 默认值是 0
EXPInit.Levels.Building=0                       ; 浮点数列表 , 获得这个等级的经验值 ， 默认值是 0

; 防御的出厂经验值
; 以下这三个列表的元素数量需要相匹配
EXPInit.Types.Defense=                          ; 防御列表 , 这些单位获得经验值 ， 默认值是 空
EXPInit.Amounts.Defense=0                       ; 浮点数列表 , 获得这个数量的经验值 ， 默认值是 0
EXPInit.Levels.Defense=0                        ; 浮点数列表 , 获得这个等级的经验值 ， 默认值是 0

; 步兵的出厂经验值
; 以下这三个列表的元素数量需要相匹配
EXPInit.Types.Infantry=                         ; 步兵列表 , 这些单位获得经验值 ， 默认值是 空
EXPInit.Amounts.Infantry=0                      ; 浮点数列表 , 获得这个数量的经验值 ， 默认值是 0
EXPInit.Levels.Infantry=0                       ; 浮点数列表 , 获得这个等级的经验值 ， 默认值是 0

; 载具的出厂经验值
; 以下这三个列表的元素数量需要相匹配
EXPInit.Types.Unit=                             ; 载具列表 , 这些单位获得经验值 ， 默认值是 空
EXPInit.Amounts.Unit=0                          ; 浮点数列表 , 获得这个数量的经验值 ， 默认值是 0
EXPInit.Levels.Unit=0                           ; 浮点数列表 , 获得这个等级的经验值 ， 默认值是 0

; 飞机的出厂经验值
; 以下这三个列表的元素数量需要相匹配
EXPInit.Types.Aircraft=                         ; 飞机列表 , 这些单位获得经验值 ， 默认值是 空
EXPInit.Amounts.Aircraft=0                      ; 浮点数列表 , 获得这个数量的经验值 ， 默认值是 0
EXPInit.Levels.Aircraft=0                       ; 浮点数列表 , 获得这个等级的经验值 ， 默认值是 0
```
