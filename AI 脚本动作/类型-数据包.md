# 类型-数据包

## 注册

位于 `rulesmd.ini`：

```ini
[DataPackTypes]
0=DataPackType1
1=DataPackType2
2=DataPackType3
```

### 注意

* 索引从 0 开始。

* 建议在 `rulesmd.ini` 本体中注册。



## 完整结构

位于 `rulesmd.ini`：

```ini
[SomeDataPackType]
IDCode=0                                        ; 整数 , 取值范围 0 ~ 65535
TechnoList.0=                                   ; Techno 列表
TechnoList.1=                                   ; Techno 列表
TechnoList.2=                                   ; Techno 列表
TechnoList.3=                                   ; Techno 列表
NumberList.0=                                   ; double 列表 , 不小于 0
NumberList.1=                                   ; double 列表 , 不小于 0
NumberList.2=                                   ; double 列表 , 不小于 0
NumberList.3=                                   ; double 列表 , 不小于 0
NumberList.4=                                   ; double 列表 , 不小于 0
NumberList.5=                                   ; double 列表 , 不小于 0
NumberList.6=                                   ; double 列表 , 不小于 0
NumberList.7=                                   ; double 列表 , 不小于 0
NumberList.8=                                   ; double 列表 , 不小于 0
NumberList.9=                                   ; double 列表 , 不小于 0
MarkList=                                       ; 标记列表
MarkListOther=                                  ; 标记列表
TeamList=                                       ; 作战小队类型列表
ScriptList=                                     ; 脚本类型列表
HasMission=yes                                  ; 布尔
MissionList=                                    ; 任务列表 , 文本 , 栗如 : Guard , Hunt
CreateCount=1                                   ; 整数 , 不小于 0
CreateCount.UseVar=0                            ; 整数 , 不小于 0
CreateCount.Same=yes                            ; 布尔
SingleTeam=yes                                  ; 布尔
SelectAll.Team=yes                              ; 布尔
SelectAll.Team.Count=1                          ; 整数 , 不小于 0
SelectAll.Script=yes                            ; 布尔
SelectAll.Script.Count=1                        ; 整数 , 不小于 0
SelectAll.Mission=yes                           ; 布尔
SelectAll.Mission.Count=1                       ; 整数 , 不小于 0
OffsetRange.0.Max=0                             ; 整数 , 不小于 0
OffsetRange.0.Min=0                             ; 整数 , 不小于 0
OffsetRange.0.Random=yes                        ; 布尔
OffsetRange.0.Self=yes                          ; 布尔
OffsetRange.1.Max=0                             ; 整数 , 不小于 0
OffsetRange.1.Min=0                             ; 整数 , 不小于 0
OffsetRange.1.Random=yes                        ; 布尔
OffsetRange.1.Self=yes                          ; 布尔
OffsetRange.2.Max=0                             ; 整数 , 不小于 0
OffsetRange.2.Min=0                             ; 整数 , 不小于 0
OffsetRange.2.Random=yes                        ; 布尔
OffsetRange.2.Self=yes                          ; 布尔
Timer.0.Step=30                                 ; 整数 , 不小于 0
Timer.0.Loop=1                                  ; 整数 , 不小于 0
Timer.0.GetNew=yes                              ; 布尔
Timer.1.Step=30                                 ; 整数 , 不小于 0
Timer.1.Loop=1                                  ; 整数 , 不小于 0
Timer.1.GetNew=yes                              ; 布尔
Timer.2.Step=30                                 ; 整数 , 不小于 0
Timer.2.Loop=1                                  ; 整数 , 不小于 0
Timer.2.GetNew=yes                              ; 布尔
Another=                                        ; 关联另一个数据包 , 虽然可以填自己 , 但是没啥意义吧
```

### 注意

* 数据包的每个属性的含义由使用这个数据包的 AI 脚本动作（或其他的什么结构）定义，这些属性本身不具有默认的含义，由于数据读取的限制，它们只规定了可以接受的数据类型。
